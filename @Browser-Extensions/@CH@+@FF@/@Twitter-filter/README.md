# Twitter User Filter

Hybrid Chrome + Firefox extension that hides tweets from specific users and keywords.

## Features

- Block users
- Block keywords
- Auto-hide tweets
- Export blocklist
- Import blocklist
- Chrome + Firefox compatible
- Sync storage
- Filter counter on screen
- Reset filter counter
- Works on x.com and twitter.com
- Fully local, no tracking

## Installation (Chrome)

1. Download folder
2. Open Chrome
3. Go to chrome://extensions/
4. Enable Developer Mode
5. Click Load Unpacked
6. Select twitter-user-filter folder

## Installation (Firefox)

1. Open Firefox
2. Go to about:debugging
3. Click This Firefox
4. Load Temporary Add-on
5. Select manifest.json

## Usage

- Open X (Twitter)
- Click extension
- Add usernames or keywords
- Tweets disappear automatically
- Export/import blocklist
- Reset filtered counter anytime

## Privacy

- No external servers
- No tracking
- No analytics
- All data stored locally in browser
