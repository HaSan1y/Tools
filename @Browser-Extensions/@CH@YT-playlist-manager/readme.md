# YT Playlist Manager

Backup and export your playlists for YouTube safely and locally.

## Features

- Scan playlists
- Export JSON
- Export CSV
- Export TXT
- Reset counter
- Chrome and Firefox compatible
- No tracking
- Local only

## How to install

### Chrome

1. Open chrome://extensions
2. Enable Developer Mode
3. Load unpacked
4. Select folder

### Firefox

1. Open about:debugging
2. This Firefox
3. Load Temporary Add-on
4. Select manifest.json

## Usage

1. Open YouTube
2. Go to playlists
3. Click extension
4. Scan playlists
5. Export

## Privacy

No data leaves your computer.
